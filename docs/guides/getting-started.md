# Getting Started

## Installation

```bash
pip install -r requirements.txt
```

## Configuration

Create a `.env` file with your configuration:

```
SECRET_KEY=your-secret-key
DATABASE_URL=postgresql://localhost/mydb
```

## Running the Application

```bash
python app.py
```

## Testing

Run the test suite:

```bash
pytest tests/
```
